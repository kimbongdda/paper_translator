Decentralized Multi-State Q-Learning for NOMA-ALOHA Systems

Xueyu Wu, Youngwook Ko, and Andy M. Tyrrell

Abstract—This paper developed a decentralized multi-state QLearning algorithm for joint slot and power level selection in NOMA-ALOHA systems, which models the problem as a Markov Decision Process (MDP) and evaluates the quality of each action by a Q-Table. In a distributed manner, the proposed algorithm aims to ﬁnd the best action strategy for each user who takes into account both the random collision with other users and the SINR threshold. In particular, novel reward function and state deﬁnition are proposed to achieve better exploration and leverage history observations, respectively. Simulation results indicate that the proposed algorithm can improve the average number of users with desired ASR compared to a number of benchmarks.

Index Terms—Multiple access control, NORA, decentralized learning, reinforcement learning

I. INTRODUCTION

Random access (RA) has became one of the main challenges because of the huge number of devices leading to serious congestion in future 6G wireless networks [1]. In a legacy four-steps grant-based (GB) RA, a handshake between the users and the base station (BS) must be proceeded before the users transmit any data bits to the BS. This process aims to allocate a dedicate resource block (RB) to each user by using the physical random access channel (PRACH) and physical downlink control channel (PDCCH) [2]. However, the performance of GB-RA protocols is limited by signalling overhead problems caused by the small payload of machine type communications (MTC) devices. As a result, high efﬁciency RA protocols are demanded in 6G. Grant free (GF) RA is one of the potential candidates to address this problem because it inherently has smaller signalling overhead and thus higher spectral efﬁciency. In particular, the users directly access an RB without handshake with the BS. A commonly used RA protocol named ALOHA was developed and formulated by [3]. In ALOHA, all the users having packets are supposed to send their data with a constant access probability at any time instance. Slotted ALOHA (SA) is one of the well known ALOHA variations. In SA, all the transmission intervals are synchronized such that all the users transmit their packets in identical slots. Theoretical analysis of the fairness and the channel utilization in SA are investigated using Markov Models developed in [4] and [5]. A lazy policy

In addition, the spectral efﬁciency is limited due to the exclusive accessibility of each RB in orthogonal random access (ORA). However, non-orthogonal multiple access (NOMA) has became a hot topic because it can enhance the spectral efﬁciency by allowing multiple users to access one RB at the same time. In order to successfully decode the signals from different users, the users using the same RB are required to adopt different transmit power in NOMA [7]. [8] modiﬁed ALOHA by combining multichannel ALOHA with the principle of NOMA. Both the theoretical and numerical analysis indicate that the NOMA-ALOHA in [8] is capable of improving the throughput. Nevertheless, the random collision between users is still a main reason for the bottleneck of throughput. Besides, the demand on channel state information (CSI) caused by the channel inversion will consume a huge amount of resources, which is not feasible for resources limited MTC scenarios. Artiﬁcial intelligence (AI), especially reinforcement learning (RL), has became an emerging technology for MAC design due to its capability of solving complex problems without a comprehensive model. [9], [10] and [11] proposed deep reinforcement learning (DRL) based algorithms for RA problems while the high complexity of DRL makes it unfeasible for MTC devices with limited memory and computation resources. Q-Learning algorithms based on Q-Table are proposed in [12] and [13] but these papers only considered collisions and the channel condition is assumed to be perfect, which is not a practical assumption. Another Q-Table based method is proposed in [14] while it only considers fading, which means capturing effect is included to deal with random collisions and extra complexity is added to the system. In [15], four single state Q-Learning algorithms are proposed, in which both random collisions and fading are considered for NOMA-ALOHA systems. Nevertheless, [15] only investigates the cases that the action selection of each user originate from a single state, which can be much different in future massive MTC (mMTC) networks. Besides, the performance of individual users is still limited when ensuring the fairness between users. 그림 2에서는 적절한 μ 값을 찾기 위해 과분포된 경우(N = 2LK)에서 서로 다른 μ 값을 갖는 제안된 알고리즘의 민감도를 조사했습니다. 다중 상태 Q-Learning NORA 알고리즘은 µ에 매우 민감하고 µ = 10−2.5일 때 원하는 ASR로 가장 높은 평균 사용자 수를 달성한다는 것을 알 수 있습니다. 이는 μ 값에 따라 결정되는 전송 실패에 대한 보상과 전송 성공에 대한 보상의 비율이 적절한 범위에 속하지 않으면 게임 초기에 실패한 행위에 대해 알고리즘이 충분히 탐색할 수 없기 때문입니다. 대조적으로, µ 값이 적절하게 설정되면 알고리즘이 사용자에게 가장 적합한 전략을 찾을 가능성이 높아집니다. 이러한 분석을 바탕으로 다음 시뮬레이션에서 제안된 다중 상태 Q-Learning NORA 알고리즘에 대해 μ를 10−2.5로 설정하여 원하는 ASR을 가진 평균 사용자 수 측면에서 성능을 최대화합니다. 그림 3은 제안된 다중 상태 Q-Learning NORA 알고리즘과 벤치마크 알고리즘의 훈련 과정을 보여준다. MTC 장치는 작은 페이로드와 산발적인 트래픽으로 인해 수렴 속도가 매우 중요합니다. 제안한 알고리즘의 성능 수렴 속도는 벤치마크 예시와 유사하지만 원하는 ASR을 가진 평균 사용자 수는 벤치마크보다 훨씬 더 높습니다. 위의 관찰은 그것이 가능하다는 것을 나타냅니다.

본 논문에서는 NORA 시스템을 위한 새로운 분산형 다중 상태 다중 에이전트 강화 학습 프레임워크가 개발되었습니다. 무작위 충돌과 페이딩이 모두 존재하는 경우 보상 및 상태 기능을 갖춘 새로운 다중 상태 Q-Leaning 알고리즘을 각 사용자가 분산 방식으로 공동 슬롯 및 전력 수준 선택을 수행하는 데 사용했습니다. 수치 결과는 제안된 알고리즘이 높은 혼잡 트래픽 하에서 원하는 ASR을 가진 평균 사용자 수 측면에서 일련의 벤치마크 예제보다 더 나은 성능을 발휘한다는 것을 확인합니다. 이는 6G의 수많은 무작위 연결에 중요합니다. 또한, 제안된 알고리즘의 우수성은 벤치마크 사례에 비해 극심한 혼잡 상황에서 더욱 증가한다. 모델 없는 분산 학습 프레임워크와 NORA 절차를 기반으로 제안된 다중 상태 Q-Learning 알고리즘은 비대칭 환경에서 자원이 제한된 MTC 장치에 대한 RA의 효율성을 향상시킬 수 있습니다.
