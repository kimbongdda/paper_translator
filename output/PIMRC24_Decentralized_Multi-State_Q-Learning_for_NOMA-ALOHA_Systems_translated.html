NOMA-ALOHA 시스템을 위한 분산형 다중 상태 Q-Learning

우 쉬에유(Xueyu Wu), 고영욱, 앤디 M. 티렐(Andy M. Tyrrell)

요약 - 이 문서에서는 NOMA-ALOHA 시스템의 공동 슬롯 및 전력 수준 선택을 위한 분산형 다중 상태 Q-학습 알고리즘을 개발했습니다. 이 알고리즘은 문제를 MDP(Markov Decision Process)로 모델링하고 Q-테이블로 각 동작의 품질을 평가합니다. 제안된 알고리즘은 분산 방식으로 다른 사용자와의 무작위 충돌과 SINR 임계값을 모두 고려하여 각 사용자에 대한 최선의 행동 전략을 찾는 것을 목표로 합니다. 특히, 각각 더 나은 탐색을 달성하고 이력 관찰을 활용하기 위해 새로운 보상 함수와 상태 정의가 제안되었습니다. 시뮬레이션 결과는 제안된 알고리즘이 여러 벤치마크에 비해 원하는 ASR을 가진 평균 사용자 수를 향상시킬 수 있음을 나타냅니다.

색인 용어 - 다중 액세스 제어, NORA, 분산 학습, 강화 학습

I. 소개

RA(Random Access)는 향후 6G 무선 네트워크에서 심각한 혼잡을 초래하는 엄청난 수의 장치로 인해 주요 과제 중 하나가 되었습니다[1]. 기존 4단계 그랜트 기반(GB) RA에서는 사용자가 BS에 데이터 비트를 전송하기 전에 사용자와 기지국(BS) 간의 핸드셰이크가 진행되어야 합니다. 이 과정은 PRACH(Physical Random Access Channel)와 PDCCH(Physical Downlink Control Channel)를 이용하여 각 사용자에게 전용 자원블록(RB)을 할당하는 것을 목표로 한다[2]. 그러나 GB-RA 프로토콜의 성능은 MTC(Machine Type Communications) 장치의 작은 페이로드로 인해 발생하는 시그널링 오버헤드 문제로 인해 제한됩니다. 이에 따라 6G에서는 고효율 RA 프로토콜이 요구된다.

GF(Grant Free) RA는 본질적으로 더 작은 시그널링 오버헤드와 더 높은 스펙트럼 효율성을 갖기 때문에 이 문제를 해결할 수 있는 잠재적 후보 중 하나입니다. 특히, 사용자는 기지국과의 Handshake 없이 RB에 직접 접속하게 된다. 일반적으로 사용되는 ALOHA라는 RA 프로토콜은 [3]에 의해 개발 및 공식화되었습니다. ALOHA에서는 패킷을 가진 모든 사용자가 언제든지 일정한 액세스 확률로 데이터를 전송해야 합니다. 슬롯형 ALOHA(SA)는 잘 알려진 ALOHA 변형 중 하나입니다. SA에서는 모든 사용자가 동일한 슬롯에서 패킷을 전송하도록 모든 전송 간격이 동기화됩니다. SA의 공정성 및 채널 활용에 대한 이론적 분석은 [4]와 [5]에서 개발된 Markov 모델을 사용하여 조사됩니다. 게으른 정책

SA에서 AAOI(시간 평균 정보 연령)를 최소화하는 것이 [6]에 의해 제안되었습니다.

또한 ORA(Orthogonal Random Access)에서는 각 RB의 배타적 접근성으로 인해 스펙트럼 효율성이 제한됩니다. 그러나 NOMA(Non-orthogonal Multiple Access)는 여러 사용자가 하나의 RB에 동시에 액세스할 수 있도록 함으로써 스펙트럼 효율성을 향상시킬 수 있다는 점에서 화제가 되었습니다. 서로 다른 사용자의 신호를 성공적으로 디코딩하려면 동일한 RB를 사용하는 사용자가 NOMA에서 서로 다른 전송 전력을 채택해야 합니다[7]. [8] 다중 채널 ALOHA와 NOMA 원리를 결합하여 ALOHA를 수정했습니다. 이론적 및 수치적 분석 모두 [8]의 NOMA-ALOHA가 처리량을 향상시킬 수 있음을 나타냅니다. 그럼에도 불구하고 사용자 간의 무작위 충돌은 여전히 ​​처리량 병목 현상의 주요 원인입니다. 게다가 채널 반전으로 인해 발생하는 채널 상태 정보(CSI)에 대한 수요는 엄청난 양의 리소스를 소비하게 되며 이는 리소스가 제한된 MTC 시나리오에서는 실현 가능하지 않습니다.

인공 지능(AI), 특히 강화 학습(RL)은 포괄적인 모델 없이 복잡한 문제를 해결할 수 있는 능력으로 인해 MAC 설계의 새로운 기술이 되었습니다. [9], [10] 및 [11]에서는 RA 문제에 대한 심층 강화 학습(DRL) 기반 알고리즘을 제안했지만 DRL의 복잡성이 높기 때문에 메모리와 계산 리소스가 제한된 MTC 장치에서는 실행 불가능합니다. Q-Table을 기반으로 한 Q-학습 알고리즘은 [12]와 [13]에서 제안되었으나 이들 논문에서는 충돌만 고려했으며 채널 상태는 완벽하다고 가정하므로 실제적인 가정은 아닙니다. 페이딩만 고려하는 또 다른 Q-테이블 기반 방법이 [14]에서 제안되었습니다. 이는 무작위 충돌을 처리하기 위해 캡처 효과가 포함되고 시스템에 추가적인 복잡성이 추가된다는 의미입니다. [15]에서는 NOMA-ALOHA 시스템에 대해 무작위 충돌과 페이딩을 모두 고려하는 4개의 단일 상태 Q-학습 알고리즘이 제안되었습니다. 그럼에도 불구하고 [15]에서는 각 사용자의 행동 선택이 단일 상태에서 발생하는 경우만 조사하며, 이는 미래의 대규모 MTC(mMTC) 네트워크에서는 크게 다를 수 있습니다. 게다가, 사용자 간의 공정성을 보장할 때 개별 사용자의 성과는 여전히 제한적입니다.

본 논문에서는 줄여서 NORA(Non-Orthogonal Random Access)라고 불리는 NOMA-ALOHA 시스템을 위한 분산형 다중 상태 Q-학습 알고리즘을 개발하여 분산된 사용자가 채널 정보 가용성 없이 개별 사용자의 전력 수준과 하위 채널 선택을 최적화하도록 개발했습니다. 본 논문의 주요 기여는 다음과 같이 요약된다.

  • 분산된 사용자가 다중 전력 레벨을 갖춘 NOMA의 기능과 중앙 집중식 액세스 제어가 없는 슬롯형 ALOHA의 기능을 활용하여 무단 무작위 액세스를 채택하는 NORA 시스템이 개발되었습니다.
  • 다른 사용자와의 상호작용 없이 패킷 처리량을 향상시키는 전략을 찾기 위해 제안된 시스템 모델과 관련된 새로운 분산형 강화학습 구조를 Markov Decision Process의 형태로 개발하였다.
  • 다중 상태 Q-Learning에 대한 새로운 보상 기능은 더 나은 탐색을 달성하도록 설계되었습니다.
  • 다중 상태 Q-Learning을 위한 NORA 작업에 해당하는 상태 공간은 이력 관찰을 활용하기 위해 개발되었습니다.
  • 시스템의 원하는 ASR을 갖는 평균 사용자 수를 실험으로 평가하고, 시뮬레이션 결과를 통해 제안하는 알고리즘이 일련의 벤치마크에 비해 원하는 ASR을 갖는 평균 사용자 수를 향상시킬 수 있음을 나타냅니다.

Ⅱ. 시스템 모델

이 섹션에서는 본 논문에서 사용된 분산형 NORA 시스템을 소개합니다. 그림 1과 같이 Kslot을 통해 BS로 패킷을 전송하는 N명의 사용자가 있습니다. \(h_{n:k}\)은 슬롯 \(k \in \{1, \dots, K\}\)을 통해 사용자 n에서 BS로의 채널 계수를 나타냅니다. 각 사용자는 \(h_{n:k} \sim \mathcal{CN}(0, \bar{\alpha}_{n:k})\)로 레일리 페이딩 채널을 경험한다고 가정합니다. 여기서 \(\bar{\alpha}_{n:k}\)은 n번째 사용자의 평균 채널 이득입니다. \(\alpha_{n:k} = |h_{n:k}|^2\)에 의한 순간 채널 이득을 나타냅니다.

사용자는 K개의 슬롯 중 하나를 무작위로 선택하고 L개의 전력 레벨 중 하나로 선택한 슬롯을 통해 패킷을 전달합니다. 슬롯과 전력 레벨의 선택은 상호 작용 및 채널 정보 없이 이루어집니다. 이 상황에서 사용자는 무허가 임의 액세스 전략을 개발하기를 원합니다. 시그널링 오버헤드를 완화하고 스펙트럼 효율성을 향상시켜야 하기 때문에 전력 도메인 NOMA 기반 ALOHA가 고려됩니다. 편의상 본 논문에서는 NORA라고 부른다. NORA 시스템에 대한 자세한 내용은 섹션 II.A에 설명되어 있습니다.

A. NORA 단계

각 전송 간격이 시작될 때 각 사용자는 송신기에 CSI가 없는 슬롯과 전력 수준을 선택합니다. n번째 사용자의 동작 \(a_n(t) \in \mathcal{A}\)은 채널 슬롯 k와 전송 전력 \(P_l\)을 포함하며 이는 다음과 같습니다.

\[a_n(t) = \begin{cases} (0,0) & \text{no transmit at step } t \\ (k, P_l) & \end{cases}\]

(식 1)

여기서 \(l \in \{1, ..., L\}\)은 전력 레벨 인덱스를 나타내고, \(k \in\) \(\{1,\ldots,K\}\)는 슬롯 인덱스를 나타내고, \(\mathcal{A}\)은 동작 공간을 나타냅니다. 정규화된 전력 레벨은 \(P_l \in (0,1)\)로 표시되며,

\[Z_{n:k,l} = \mathbb{1}(a_n = (k, P_l)).\]

(식 2)

슬롯 k에서 BS에 의해 수신된 신호는 다음과 같이 주어진다.

\[y_k = \sum_{l=1}^{L} \sum_{n=1}^{N} \sqrt{P_l} h_{n;k} s_n Z_{n;k,l} + w_k\]

(식 3)

여기서 \(w_k \in \mathcal{CN}(0, N_0)\)과 \(s_n\)은 각각 슬롯 k의 AWGN(Additive White Gaussian Noise)과 변조된 기호를 나타냅니다. (3)에 따르면, 다수의 사용자가 동일한 슬롯 k에서 무작위로 경쟁할 때 NOMA 전송에서 연속 간섭 제거(SIC)에 의해 신호를 디코딩하는 것이 가능할 수 있습니다.

B. 문제 설명

개별 사용자의 처리량을 최대화하는 분산형 무허가 NORA 알고리즘이 제안됩니다. ASRn은 n번째 사용자의 평균 성공률(ASR)[15]을 나타내며 다음과 같이 정의됩니다.

\[ASR_n = \mathbb{E}[Flag_n]. \tag{6}\]

ASR은 특정 사용자의 전략에 대해 n번째 사용자가 달성한 성공적인 전송의 평균 횟수를 측정합니다. 알고리즘 설계는 사용자 간의 공정성을 향상시키는 것이 바람직합니다. 즉, 각 사용자는 ASRn을 처리량 임계값보다 크거나 동일하게 만들어야 합니다. 따라서 이 기법의 성능은 ASRn이 원하는 ASR의 임계값 ASR0보다 큰 평균 사용자 수로 분석되며 다음과 같이 공식화됩니다.

\[N_{users} = \frac{1}{K} \sum_{n=1}^{N} \mathbb{1}(ASR_n > ASR_0). \tag{7}\]

각 사용자의 행동 전략은 서로 알려지지 않은 다른 사용자의 행동 전략에 의해 영향을 받기 때문에 각 사용자는 분산된 방식으로 자신과 환경 간의 상호 작용을 통해 학습하는 것을 목표로 합니다. 원하는 ASR을 가진 평균 사용자 수를 향상시키기 위해 III장에서는 개별 사용자의 행동 전략을 학습하는 분산형 Q-Learning 지원 NORA 알고리즘을 제안합니다.

III. 제안된 다중 상태 Q-학습 알고리즘

본 논문에서는 슬롯 및 전력 수준 선택 프로세스를 MDP로 모델링합니다. MDP에서는 에이전트가 환경을 관찰하여 상태를 획득하고, 관찰된 상태에 따라 조치를 취하고, 각 단계에서 보상을 받습니다. Q-학습은 MDP를 다루는 가장 일반적으로 사용되는 강화 학습 알고리즘 중 하나입니다[16]. 본 논문에서 채택한 Q-학습 모델은 본 논문에서 Q 테이블로 구현된 액션 값 함수를 기반으로 각 사용자를 액션(k, Pl) 중 하나를 선택하는 에이전트로 모델링합니다. 각 사용자의 Q-테이블은 아래 방정식에 따라 갱신됩니다.

\[Q_n(s_n(t), a_n(t)) \leftarrow Q_n(s_n(t), a_n(t)) + \alpha \delta_n(t)\]

(식 8)

알고리즘 1 다중 상태 Q-학습 NORA

Initialization: Initialize all the Q-values with zeros, t = 0,
an(0) = (0, 0), Sn(1) = (0, 0), T = 5000, α = 0.1, γ =
0.05
while t < T do
   t ← t + 1
   an(t) ← arg maxa Q(Sn(t), a)
   if an(t) = (0, 0) then
       Rn(t + 1) ← 0
   else
       Access the channel according to an(t) and observe
Flagn(t) through the feedback signal from the BS
       if Flagn(t) = 1 then
          Rn(t + 1) ← 1
       else
          Rn(t + 1) ← −µ
       end if
   end if
   Sn(t + 1) ← an(t)
   Qn(Sn(t), an(t)) ← Qn(Sn(t), an(t)) + αδn(t)

여기서 sn(t), an(t) 및 α는 각각 상태, 동작 및 학습률을 나타내며 시간차(TD) 오류 δn(t)는 다음과 같이 지정됩니다.

동안 종료

\[\delta_n(t) = G_n(t) - Q_n(s_n(t), a_n(t))\]
\[= R_n(t) + \gamma \max_a Q_n(s_n(t+1), a) - Q_n(s_n(t), a_n(t))\]

(식 9)

여기서 Rn(t) 및 γ는 각각 즉각적인 보상 및 할인 요소를 나타냅니다.

본 논문에서는 섹션 II.B에서 공식화한 NORA 문제를 해결하기 위해 특별히 설계된 상태 및 보상 정의를 갖춘 새로운 Q-학습 알고리즘을 제안합니다. 알고리즘은 각 사용자의 개별 전략을 보여줍니다. 이는 모든 사용자에게 동일하다는 것을 의미합니다.

사용자가 더 나은 결정을 내리기 위해 기록 관찰을 활용할 수 있도록 MDP의 상태는 마지막 단계 t − 1에서 수행된 작업으로 설정됩니다.

\[s_n(t) \triangleq a_n(t-1) \tag{10}\]

많은 논문 [17], [15], [10], [14]에서 전송 실패에 대한 보상은 -1 또는 0으로 설계되었으며 시스템 모델에서 높은 수렴 속도를 얻습니다. 그럼에도 불구하고 성공적인 전송과 실패 전송에 대한 보상이 동일한 절대값을 갖는 경우 Q 값의 일부가 과소평가될 수 있으며, 이로 인해 해당 사용자는 항상 성공하고 다른 사용자는 항상 실패하게 됩니다. 이러한 단점을 해결하기 위해 실패한 전송에 대한 보상의 절대값인 μ를 성공적인 전송에 대한 보상보다 작게 만듭니다. 일반화의 손실 없이 단순성을 위해 성공적인 전송에 대한 보상은 μ*가 실제로 전송 실패 보상 크기의 비율을 나타내도록 1로 설정됩니다.

그림 2: N=24,~K=4,~L=3일 때 전송 실패에 대한 보상 절대값 \(\mu\)이 원하는 ASR을 가진 평균 사용자 수에 미치는 영향. 성공적인 전송에 대한 보상은 1입니다.

성공적인 전송 보상 규모. MDP의 보상 정의는 다음과 같이 설계되었습니다.

\[R_n(t+1) \triangleq \begin{cases} 1 & \text{if } Flag_n(t) = 1\\ 0 & \text{if } a_n(t) = (0,0)\\ -\mu & \text{if } a_n(t) \neq (0,0), \ Flag_n(t) = 0 \end{cases}\]

(식 11)

여기서 \(\mu > 0\) 은 다음과 같습니다.

\[\mathbb{E}[R_n] = \mathbb{E}[ASR_n] - \mu(1 - \mathbb{E}[ASR_n]) \tag{12}\]

(식 12)에 따르면 n번째 사용자 \(\mathbb{E}[R_n]\)가 받는 평균 보상은 \(ASR_n\)에 따라 증가합니다. 결과적으로 보상을 극대화하는 알고리즘은 원하는 ASR을 가진 평균 사용자 수를 향상시킵니다. 학습 모델과 알고리즘은 (11)을 기반으로 아래에 제시됩니다. 알고리즘 1은 다중 상태 Q-Learning 지원 NORA 알고리즘을 보여줍니다.

IV. 수치 결과 및 토론

이 섹션에서는 제안된 분산형 NORA 알고리즘의 수치 결과 및 분석을 제공합니다. 비교 목적을 위해 시스템에서 4가지 참조 체계가 시뮬레이션됩니다.

  • NOMA가 있는 슬롯형 ALOHA [8].
  • RL-NORA 가속-GA [15].
  • RL-NORA 가속 - \(\epsilon\) -GA [15].
  • woSDC-BAP-QL [17].

모든 시뮬레이션의 경우 \(\bar{\alpha}_{n;k}=10dB, \forall n\in\{1,\cdots,N\}, \forall k\in\{1,\cdots,K\}, \Gamma=-3dB, L=3 \text{ with } P_1=0.04, P_2=0.16, P_3=0.8\) 입니다. 이 백서에서는 네 가지 시뮬레이션 시나리오를 조사합니다.

• 실패한 전송에 대한 보상의 절대값인 \(\mu\)이 제안된 알고리즘의 원하는 ASR을 가진 평균 사용자 수에 미치는 영향.

그림 3: 수렴 속성, N=24, K=4, L=3, \(\mu=10^{-2.5}\) . 비교를 위해 NOMA[8], RL-NORA Acceleration-GA[15], RL-NORA Acceleration-\(\epsilon\)-GA[15] 및 woSDC-BAP-QL[17]을 갖춘 슬롯형 ALOHA가 표시됩니다.

  • 제안된 알고리즘과 벤치마크 방식의 다양한 사용자 수 N에 대해 원하는 ASR을 가진 평균 사용자 수입니다.
  • 제안한 알고리즘의 융합 특성.
  • 제안된 알고리즘의 원하는 ASR을 가진 평균 사용자 수에 슬롯 개수 K가 미치는 영향.

그림 2에서는 \(\mu\)에 적합한 값을 찾기 위해 과분산된 경우(N = 2LK)에서 다양한 \(\mu\) 값을 갖는 제안된 알고리즘의 민감도를 조사했습니다. 다중 상태 Q-Learning NORA 알고리즘은 \(\mu\)에 대해 매우 민감하고, \(\mu = 10^{-2.5}\)에 대해 원하는 ASR로 가장 높은 평균 사용자 수를 달성한다는 것을 알 수 있습니다. 이는 \(\mu\)의 값에 의해 결정되는 전송 실패에 대한 보상과 전송 성공에 대한 보상의 비율이 적절한 범위에 있지 않으면 게임 초기에 실패한 행위에 대해 알고리즘이 충분한 탐색을 수행할 수 없기 때문입니다. 반면, \(\mu\)의 값을 적절하게 설정하면 알고리즘이 사용자에게 가장 적합한 전략을 찾을 가능성이 높아집니다. 이러한 분석을 바탕으로 다음 시뮬레이션에서 제안된 다중 상태 Q-Learning NORA 알고리즘에 대해 \(\mu\)을 (10)^{-2.5}$로 설정하여 원하는 ASR을 가진 평균 사용자 수 측면에서 성능을 극대화했습니다.

그림 3은 제안된 다중 상태 Q-Learning NORA 알고리즘과 벤치마크 알고리즘의 훈련 과정을 보여준다. MTC 장치의 경우 페이로드가 작고 산발적인 트래픽이 발생하기 때문에 수렴 속도가 매우 중요합니다. 제안한 알고리즘의 성능 수렴 속도는 벤치마크 예시와 유사하지만 원하는 ASR을 가진 평균 사용자 수는 벤치마크보다 훨씬 더 높습니다. 위의 관찰은 그것이 가능하다는 것을 나타냅니다.

그림 4: \(\mu=10^{-2.5},~K=4,~L=3\)일 때 원하는 ASR이 있는 평균 사용자 수에 대한 사용자 수 N의 영향. 비교를 위해 NOMA[8], RL-NORA Acceleration-GA[15], RL-NORA Acceleration-\(\epsilon\)-GA[15] 및 woSDC-BAP-QL[17]을 갖춘 슬롯형 ALOHA가 표시됩니다.

MTC 장치가 제안된 알고리즘을 배포하도록 합니다.

그림 4에서는 제안된 알고리즘과 벤치마크 알고리즘의 원하는 ASR을 갖는 평균 사용자 수에 대한 다양한 사용자 수 N의 영향을 시뮬레이션합니다. 모든 알고리즘의 성능은 N에 따라 먼저 상승한 다음 저하됩니다. 이는 N이 LK보다 작을 때 소수의 사용자가 성능 병목 현상을 일으키는 주요 원인이기 때문입니다. 네 가지 벤치마크 방식 모두 N > LK일 때 성능 저하가 발생하는 반면 제안된 다중 상태 Q-Learning NORA 알고리즘은 높은 수준에서 성능을 유지합니다. 이는 사용자 수 N이 행동 수(LK)보다 높을 때 알고리즘이 이력 관찰을 충분히 활용할 수 없으면 충돌을 완화하고 수신 신호 SINR을 최소 SINR 임계값으로 만드는 전략을 찾기가 어렵기 때문입니다. 제안된 알고리즘은 LK < N < 2LK일 때 벤치마크와 비교하여 원하는 ASR을 가진 가장 높은 평균 사용자 수를 달성합니다.

그림 5에서는 제안된 알고리즘과 벤치마크 알고리즘의 원하는 ASR을 가진 평균 사용자 수에 대한 다양한 하위 채널 수 K의 효과가 시뮬레이션됩니다. 제안한 알고리즘의 원하는 ASR을 갖는 평균 사용자 수는 하위 채널 수가 증가함에 따라 감소합니다. 이는 K가 증가하면 Q-table의 크기가 증가하여 탐색 효율성이 저하되기 때문입니다. 그럼에도 불구하고 다중 상태 Q-Learning 알고리즘은 하위 채널 수에 관계없이 모든 벤치마크보다 원하는 ASR을 가진 평균 사용자 수를 더 높게 달성하여 제안된 알고리즘의 우수한 확장성을 보여줍니다.

그림 5: \(\mu=10^{-2.5},\,L=3,\,N=2KL\)일 때 원하는 ASR을 가진 평균 사용자 수에 대한 하위 채널 수 K의 영향. 비교를 위해 NOMA[8], RL-NORA Acceleration-GA[15], RL-NORA Acceleration-\(\epsilon\)-GA[15] 및 woSDC-BAP-QL[17]을 갖춘 슬롯형 ALOHA가 표시됩니다.

V. 결론

본 논문에서는 NORA 시스템을 위한 새로운 분산형 다중 상태 다중 에이전트 강화 학습 프레임워크가 개발되었습니다. 무작위 충돌과 페이딩이 모두 존재하는 경우 보상 및 상태 기능을 갖춘 새로운 다중 상태 Q-Leaning 알고리즘을 각 사용자가 분산 방식으로 공동 슬롯 및 전력 수준 선택을 수행하는 데 사용했습니다. 수치 결과는 제안된 알고리즘이 높은 혼잡 트래픽 하에서 원하는 ASR을 가진 평균 사용자 수 측면에서 일련의 벤치마크 예제보다 더 나은 성능을 발휘함을 검증합니다. 이는 6G의 수많은 무작위 연결에 중요합니다. 또한, 제안된 알고리즘의 우수성은 벤치마크 사례에 비해 극심한 혼잡 상황에서 더욱 증가한다. 모델 없는 분산 학습 프레임워크와 NORA 절차를 기반으로 제안된 다중 상태 Q-Learning 알고리즘은 비대칭 환경에서 자원이 제한된 MTC 장치에 대한 RA의 효율성을 향상시킬 수 있습니다.
