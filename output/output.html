Decentralized Multi-State Q-Learning for NOMA-ALOHA Systems Xueyu Wu, Youngwook Ko, and Andy M. Tyrrell Abstract —This paper developed a decentralized multi-state Q- Learning algorithm for joint slot and power level selection in NOMA-ALOHA systems, which models the problem as a Markov Decision Process (MDP) and evaluates the quality of each action by aQ-Table. In a distributed manner, the proposed algorithm aims to ﬁnd the best action strategy for each user who takes into account both the random collision with other users and the SINR threshold. In particular, novel reward function and state deﬁnition are proposed to achieve better exploration and leverage history observations, respectively. Simulation results indicate that the proposed algorithm can improve the average number of users with desired ASR compared to a number of benchmarks. Index Terms —Multiple access control, NORA, decentralized learning, reinforcement learning I. I NTRODUCTION Random access (RA) has became one of the main challenges because of the huge number of devices leading to serious congestion in future 6G wireless networks [1]. In a legacy four-steps grant-based (GB) RA, a handshake between the users and the base station (BS) must be proceeded before the users transmit any data bits to the BS. This process aims to allocate a dedicate resource block (RB) to each user by using the physical random access channel (PRACH) and physical downlink control channel (PDCCH) [2]. However, the performance of GB-RA protocols is limited by signalling over- head problems caused by the small payload of machine type communications (MTC) devices. As a result, high efﬁciency RA protocols are demanded in 6G. Grant free (GF) RA is one of the potential candidates to address this problem because it inherently has smaller signalling overhead and thus higher spectral efﬁciency. In particular, the users directly access an RB without handshake with the BS. A commonly used RA protocol named ALOHA was developed and formulated by [3]. In ALOHA, all the users having packets are supposed to send their data with a constant access probability at any time instance. Slotted ALOHA (SA) is one of the well known ALOHA variations. In SA, all the transmission intervals are synchronized such that all the users transmit their packets in identical slots. Theoretical analysis of the fairness and the channel utilization in SA are investigated using Markov Models developed in [4] and [5]. A lazy policy Xueyu Wu, Youngwook Ko and Andy M. In addition, the spectral efﬁciency is limited due to the ex- clusive accessibility of each RB in orthogonal random access (ORA). However, non-orthogonal multiple access (NOMA) has became a hot topic because it can enhance the spectral efﬁ- ciency by allowing multiple users to access one RB at the same time. In order to successfully decode the signals from different users, the users using the same RB are required to adopt different transmit power in NOMA [7]. [8] modiﬁed ALOHA by combining multichannel ALOHA with the principle of NOMA. Both the theoretical and numerical analysis indicate that the NOMA-ALOHA in [8] is capable of improving the throughput. Nevertheless, the random collision between users is still a main reason for the bottleneck of throughput. Besides, the demand on channel state information (CSI) caused by the channel inversion will consume a huge amount of resources, which is not feasible for resources limited MTC scenarios. Artiﬁcial intelligence (AI), especially reinforcement learn- ing (RL), has became an emerging technology for MAC design due to its capability of solving complex problems without a comprehensive model. [9], [10] and [11] proposed deep rein- forcement learning (DRL) based algorithms for RA problems while the high complexity of DRL makes it unfeasible for MTC devices with limited memory and computation resources. Q-Learning algorithms based on Q-Table are proposed in [12] and [13] but these papers only considered collisions and the channel condition is assumed to be perfect, which is not a practical assumption. Another Q-Table based method is proposed in [14] while it only considers fading, which means capturing effect is included to deal with random collisions and extra complexity is added to the system. In [15], four single stateQ-Learning algorithms are proposed, in which both ran- dom collisions and fading are considered for NOMA-ALOHA systems. Nevertheless, [15] only investigates the cases that the action selection of each user originate from a single state, which can be much different in future massive MTC (mMTC) networks. Besides, the performance of individual users is still limited when ensuring the fairness between users. The main contributions of this paper are summarized as following. •A NORA system where distributed users adopt grant- free random access leveraging the features from NOMA with multiple power levels and slotted ALOHA with no centralized access control is developed. •A new decentralized reinforcement learning structure associated with the proposed system model is developed to ﬁnd a strategy of improving the packet throughput under no interaction with other users, in the form of Markov Decision Process. •A novel reward function for the multi-state Q-Learning is designed to achieve better exploration. •A state space corresponding to the NORA actions for the multi-state Q-Learning is developed to leverage the history observations. •Average number of users with desired ASR of the system is evaluated by experiments, and the simulation results present that the proposed algorithm can improve the average number of users with desired ASR compared to a set of benchmarks. III. P ROPOSED MULTI-STATE Q-LEARNING ALGORITHM The slot and power level selecting process is modelled as a MDP in this paper. In a MDP, the agent acquires its state by observing the environment, takes action according to the observed state, and then receives reward at each step. Q- Learning is one of the most commonly used reinforcement learning algorithm tackling MDP [ 16]. The Q-Learning model adopted in this paper models each user as an agent who selects one of the actions (k, P l)base on the action value function, which is implemented by a Q table in this paper. C ONCLUSION In this paper, a novel decentralized multi-state multi-agent reinforcement learning framework for NORA systems was developed. With the presence of both random collision and fading, the novel multi-state Q-Leaning algorithm with reward and state functions were used for each user to make the joint slot and power level selection in a distributed manner. Numerical results verify that the proposed algorithm performs better than a set of benchmark examples in terms of average number of users with desired ASR under high congestion trafﬁcs. This is important for the huge number of random connectivities in 6G. Moreover, the superiority of the proposed algorithm increases more in extreme congestion condition compared to the benchmark examples. Powered by the model- free decentralized learning framework and the NORA pro- cedure, the proposed multi-state Q-Learning algorithm can improve the efﬁciency of RA for resource limited MTC devices in asymmetric environment.
